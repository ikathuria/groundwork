# Copper Sulfate Pentahydrate

## Properties

Copper sulfate pentahydrate is a common copper source for electroplating bath chemistry. In this hypothesis it is a procurement-relevant reagent for plated copper routes, not for paste-only routes.

## Common Uses

- Copper electroplating bath component.
- Supplier-traceable input for [[../methods/nickel-copper-plated-silicon-metallization]].

## Alternatives

- Commercial photovoltaic copper plating chemistries from specialty suppliers.
- Other copper salts depending on bath formulation.

## Gotchas

- A catalog page is not a process recipe.
- Bath additives, pH, current density, agitation, and contamination control still need protocol definition.
