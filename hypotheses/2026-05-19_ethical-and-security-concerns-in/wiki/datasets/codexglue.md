CodeXGLUE is a benchmark suite for machine learning on code, covering tasks including code completion, code-to-code translation, and code search. Used in [[sources/2025-anon-retriever-backdoors-code]] to evaluate backdoor attacks on retrieval-augmented code generation (RACG) systems.
