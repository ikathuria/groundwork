# ELISA (enzyme-linked immunosorbent assay)

## What it measures / produces

Plate-based immunoassay using capture + detection antibodies and an enzymatic readout (HRP/AP) to quantify analyte concentration in solution. Output: analyte concentration (e.g., RANKL, OPG, OCN) via standard curve.

## When to use it

Use ELISA for soluble protein analytes (secreted osteogenic factors, RANKL/OPG ratio, OCN). Used in [[../sources/protocol-2025-amin-murine-osteoblast-and-osteoclast-co-cult]] for RANKL/OPG quantification (with the practical note: dilute OPG samples to keep within linear range) and [[../sources/2024-diez-escudero-in-vitro-evaluation-of-pore-size-graded-bone]] for OCN at day 14.

## Step-level notes

- Run standard curve every plate.
- Dilute samples to fall in the linear region of the standard curve [[../sources/protocol-2025-amin-murine-osteoblast-and-osteoclast-co-cult]].
