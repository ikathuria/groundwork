# Collagen-glycosaminoglycan (CG) scaffold

Freeze-dried composite of type I collagen + chondroitin-6-sulphate, the canonical Yannas/O'Brien lyophilised scaffold platform for cartilage/bone/skin regeneration. The Murphy 2010 study used it as the model system for the foundational pore-size work cited throughout this wiki.

## Properties

- **Composition:** ~95–99 wt% collagen + 1–5 wt% chondroitin-6-sulphate, freeze-dried then dehydrothermally crosslinked.
- **Pore size:** controlled by freezing temperature (–10 to –40 °C → ~325 to ~85 µm).

## Common uses in this hypothesis space

- Reference scaffold in [[../sources/2010-murphy-understanding-the-effect-of-mean-pore-size-on]] for pore-size-effect studies.

## Suppliers / catalog

In-house from purified collagen + chondroitin-6-sulphate; not a commodity catalog item.

## Alternatives

- [[../reagents/collagen]] alone — no GAG.
- [[../reagents/gelatin]]-based scaffolds — easier handling, less biomimetic.

## Gotchas / things to watch for

- Pore size set by freezing protocol; reproducibility requires controlled-rate freezer.
- Crosslinking method (DHT vs EDC) shifts mechanical properties.
